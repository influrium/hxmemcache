# hxmemcache

Memcache client library on Haxe

## Supported targets

- neko
- hashlink
- php
- python
- cpp
- java
- jvm

All targets are tested [![Build Status](https://travis-ci.org/influrium/hxmemcache.svg?branch=master)](https://travis-ci.org/influrium/hxmemcache)

## PHP target

Need extensions:

- mbstring
- socket
